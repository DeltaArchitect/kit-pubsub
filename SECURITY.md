## Reporting a Vulnerability

We take the security of `kit-pubsub` seriously. If you believe you have found a security vulnerability, please report it to us as described below.

**Please do not report security vulnerabilities through public GitHub issues.**

### Private Reporting

You can report sensitive security issues via:

1.  **Email**: Send an email to [YOUR_EMAIL_HERE] (please replace this).
2.  **GitHub Security Advisories**: If enabled, use the "Report a vulnerability" button in the Security tab.

### Process

1.  We will acknowledge your report within 48 hours.
2.  We will investigate the issue and communicate our findings.
3.  We will work with you to release a fix and a security advisory.

Thank you for improving the security of `kit-pubsub`.
